# job4j_design
В этом уровне ты узнаешь, что внутри каждой из коллекций и почему одна коллекция работает быстрее, чем другая. Познакомишься с анализом алгоритмов. Узнаешь, как работают самые важные алгоритмы. Познакомишься с концепциями ввода-вывода. Научишься использовать базы данных и хранить ценную информацию в них.
В этом проекте рассмотрены: 
- Структуры данных и алгоритмы
- Ввод-вывод
- Работа с базами данных SQL, JDBC
- Работа со сборщиком мусора Garbage Collection
- Объектно-ориентированный дизайн
